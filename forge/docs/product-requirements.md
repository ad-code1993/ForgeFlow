# Product Requirements

## Problem

Users frequently recreate resumes from scratch.

Formatting is lost when moving between tools.

---

## Solution

Upload a DOCX resume.

Automatically convert it into:

- structured data
- editable sections
- reusable templates

---

## Target Users

### Students

Need quick resume creation.

### Developers

Need multiple resume versions.

### Freelancers

Need client-specific resumes.

### Professionals

Need ATS-friendly resumes.

---

## Core Value

Transform static resumes into reusable assets.
